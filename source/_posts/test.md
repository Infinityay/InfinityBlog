## Hello, Git

## git 本地操作

`git init` 在当前文件夹创建git仓库

`git add . ` 添加所有工作区文件到缓冲区

`git status` 获取仓库状态

`git log` 查看最近提交日志

`git commit -m "输入你的备注，关于这次提交的信息"`

## git 远程操作

`git push origin main`  推送当地git库到github

`git remote -v`  查看远程库的信息

`git clone "远程仓库地址"`  将远程仓库克隆到当前文件夹
